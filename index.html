3. ページ別・機能詳細（全自動化の全貌）

① 申込・決済・認証システム

  • Amazon SES 高到達率メール認証: スパム・不正登録を防ぎ、Gmail等へ確実に通知を届けます。
  • Stripe サブスク自動決済: 月額500円/758円等のコース別自動課金。カード更新案内、再決済成功時の自動復帰まで無人化。
  • マルチステップフォーム: 離脱率を極限まで下げた申込導線。

② 動画視聴・ドリップ配信

  • 段階的自動解放: 入会日に合わせ「月に1本」または「月に2本」と自動で動画が増える独自ロジックを実装。
  • コンテンツ保護: 契約者以外は動画URLを直接叩いても視聴不可。不正コピーを防止。
  • プッシュ通知(PWA): 動画追加時、スマホへ直接「ピコーン」と通知を送信。再訪率を最大化。

③ 24時間 AI顧客サポート

  • Google AI Gemini API連携: 専門知識を学習したAIが24時間、お客様の質問に即座に回答。
  • プロンプト管理機能: 管理画面からAIの口調や知識範囲をいつでも書き換え可能。
  • FAQ併記: AI回答に加え、重要なQAは固定表示し信頼性を担保。

④ 会員・管理者用ダッシュボード

  • ユーザー管理: パスワード、メール、プラン変更、解約（セルフ化）をユーザー自身で完結。
  • 動画管理: 動画の追加、公開設定、ドラッグ＆ドロップによる表示順入れ替えが可能。
  • BASE連携: 必要な関連商品をBASEで即購入できるスムーズな外部リンク設置。

⑤ 広告・解析・マーケティング

  • Meta API Conversions API (CAPI): Cookieに頼らない正確な広告計測により、獲得単価を自動で最適化。
  • GA4データ分析: 「どこで離脱したか」を可視化。定期的なLPの文言・画像修正に活用。
